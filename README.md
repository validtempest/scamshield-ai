# ScamShield AI

ScamShield AI is an AI-based scam message detection system using NLP and Machine Learning.

## Features
- Scam / Safe Detection
- Confidence Score
- Risk Analysis
- Scam Category
- History Detection
- Dashboard Analytics

## Tech Stack
- Python
- Flask
- SQLite
- Scikit-learn
- TF-IDF
- Naive Bayes
- TailwindCSS

## AI Workflow
Input Message
→ NLP Preprocessing
→ TF-IDF Vectorization
→ Naive Bayes Classification
→ Risk Analysis
→ Result Display

## Installation

```bash
pip install -r requirements.txt
python app.py
```
